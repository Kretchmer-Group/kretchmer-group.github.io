title: Open-Source Software for Real-Time Electronic Structure Theory
description: We develop open-source software for real-time electronic structure theory. Most code can be found in the [TiDES](https://github.com/jskretchmer/TIDES) (Time-Dependent Electronic Structure) code, backended with the open-source electronic structure library PySCF and currently equipped with the ability to perform dynamics calculations using Density Functional Theory and Hartree Fock, with TiDES modularity allowing for additional calculations to be performed. [Real-Time Density Matrix Embedding Theory](https://github.com/jskretchmer/real_time_pDMET) is also available. More information can be found under [Software](https://kretchmergroup.chemistry.gatech.edu/software/).
